== Description ==

 Simple Checkout allows merchants to accept payments natively on their ecommerce store. 
 The card details are harvested inside of an elegant iframe pop-up, which means that the solution remains always up-to-date with standards such as PCI-DSS and 3D-Secure.

The plugin does not support installments.

Based on original plugin "Viva Wallet Woocommerce Payment Gateway By emspace.gr" [https://wordpress.org/plugins/woo-payment-gateway-viva-wallet/] and "Viva Payments â€“ Viva Wallet WooCommerce Payment Gateway

== Installation ==

Just follow the standard [WordPress plugin installation procedure](http://codex.wordpress.org/Managing_Plugins).

The instructions for setting up the plugin are simmilar to these [instructions](https://plugins.svn.wordpress.org/woo-payment-gateway-for-vivapayments/assets/instructions.pdf)



= How =
Just follow the [instructions](https://plugins.svn.wordpress.org/woo-payment-gateway-for-vivapayments/assets/instructions.pdf) to create a demo or live account in Viva dashboard and then use the necessary credentials in the Woocommerce payment gateway options.
